# 霍兰德职业兴趣参考表

> 本文档是「职准星」六问兴趣测评的综合推理参考资料。
> 用于模块 C（综合推理）的参考上下文，以及测评完成后向用户展示的解读材料。
> 基于霍兰德职业兴趣理论（Holland Codes / RIASEC）。

## 一、霍兰德六型人格详细释义

### R 型 · 现实型（Realistic）

| 维度 | 描述 |
|------|------|
| **核心特征** | 喜欢动手操作、使用工具和机器、与物打交道 |
| **性格倾向** | 务实、稳重、直率、机械能力、体力活动 |
| **偏好环境** | 户外工作、工厂、工地、实验室、技术车间 |
| **典型职业** | 工程师、技术员、机械师、电工、建筑师、农业技术员 |
| **校招岗位** | 技术管培生、工程助理、生产管理、质检助理 |
| **Q1 线索** | "搞定一个技术问题"、"做出一个东西" |
| **Q3 线索** | 倾向独处充电（偏 I/R 端） |
| **Q5 线索** | 钻研技术/工具/机械原理 |

### I 型 · 研究型（Investigative）

| 维度 | 描述 |
|------|------|
| **核心特征** | 喜欢分析、研究、解决复杂问题、探索未知 |
| **性格倾向** | 理性、独立、好奇、善于分析、学术倾向 |
| **偏好环境** | 实验室、研究所、大学、数据分析部门、咨询公司 |
| **典型职业** | 研究员、数据分析师、咨询顾问、科学家、市场研究员 |
| **校招岗位** | 数据分析助理、商业分析实习生、咨询助理、市场调研 |
| **Q1 线索** | "研究一个问题"、"分析数据找出规律" |
| **Q3 线索** | 偏好独处充电 |
| **Q5 线索** | 钻研原理、逻辑推理、数据分析 |

### A 型 · 艺术型（Artistic）

| 维度 | 描述 |
|------|------|
| **核心特征** | 喜欢创作、设计、自我表达、追求美感 |
| **性格倾向** | 感性、创新、自由、想象力丰富、不拘一格 |
| **偏好环境** | 工作室、设计公司、媒体、广告公司、剧场 |
| **典型职业** | 设计师、文案策划、内容运营、视频编导、插画师 |
| **校招岗位** | 内容运营、新媒体编辑、设计助理、策划助理 |
| **Q1 线索** | "创作了一个作品"、"设计了一个方案" |
| **Q4 线索** | 偏好自由探索、没有标准答案 |
| **Q5 线索** | 钻研创作/设计/写作/表达 |

### S 型 · 社会型（Social）

| 维度 | 描述 |
|------|------|
| **核心特征** | 喜欢帮助、教导、服务他人、与人合作 |
| **性格倾向** | 友善、共情、耐心、善于沟通、乐于助人 |
| **偏好环境** | 学校、医院、公益组织、HR 部门、客服中心 |
| **典型职业** | 教师、HR、培训师、社工、心理咨询师、客服经理 |
| **校招岗位** | 人力资源专员、培训助理、行政助理、客服管培生 |
| **Q1 线索** | "帮别人解决了问题"、"教导他人" |
| **Q3 线索** | 偏好社交充电（偏 S/E 端） |
| **Q5 线索** | 钻研帮人解决问题、服务他人 |

### E 型 · 企业型（Enterprising）

| 维度 | 描述 |
|------|------|
| **核心特征** | 喜欢说服、领导、管理、追求影响力与成就 |
| **性格倾向** | 自信、野心、外向、有说服力、善于社交 |
| **偏好环境** | 公司管理层、销售部、创业公司、政府机构 |
| **典型职业** | 销售经理、项目经理、创业者、市场总监、公务员 |
| **校招岗位** | 管培生、市场营销助理、销售助理、招商助理 |
| **Q1 线索** | "说服团队接受我的想法"、"带领团队达成目标" |
| **Q3 线索** | 偏好社交充电 |
| **Q5 线索** | 钻研组织管理/商业/赚钱 |

### C 型 · 常规型（Conventional）

| 维度 | 描述 |
|------|------|
| **核心特征** | 喜欢整理、归档、按规则和流程做事、注重细节 |
| **性格倾向** | 细心、有条理、保守、服从、执行力强 |
| **偏好环境** | 办公室、财务部、行政部、档案馆、银行柜台 |
| **典型职业** | 会计、审计、行政专员、档案管理员、银行柜员 |
| **校招岗位** | 行政助理、财务助理、采购助理、运营专员 |
| **Q1 线索** | "整理了一堆混乱的数据"、"按流程完成了任务" |
| **Q4 线索** | 偏好规则明确、流程清晰 |
| **Q5 线索** | 钻研整理/归档/优化流程 |

## 二、六型组合解读（霍兰德三字码）

| 三字码 | 类型组合 | 典型特质 | 推荐方向 |
|--------|---------|---------|---------|
| SEC | 社会-企业-常规 | 善于沟通协调，有管理能力，做事有条理 | HR、行政、管理培训 |
| SIE | 社会-研究-企业 | 善于分析人的需求，有咨询能力 | HRBP、培训发展 |
| ESC | 企业-社会-常规 | 有领导力，善沟通，做事严谨 | 管培生、项目经理 |
| ECS | 企业-常规-社会 | 有商业头脑，细致，善社交 | 销售管理、市场运营 |
| RIA | 现实-研究-艺术 | 动手能力强，有创造力，善于分析 | 技术研发、产品设计 |
| RIS | 现实-研究-社会 | 技术能力强，乐于助人 | 技术培训、技术支持 |
| AIS | 艺术-研究-社会 | 有创意，善于分析，喜欢与人交流 | 内容运营、品牌策划 |
| AEI | 艺术-企业-研究 | 创意+商业头脑+分析能力 | 产品经理、营销策划 |
| CSE | 常规-社会-企业 | 做事严谨，善沟通，有管理潜质 | 行政主管、财务管培 |

## 四、六问测评与霍兰德维度的对应关系

| 问题 | 节点名 | 分析维度 | 识别逻辑 |
|------|--------|---------|---------|
| Q1 | 成就感 | 综合判断（30%权重） | 任务完成→R/I；帮人→S；说服→E；创作→A；整理→C |
| Q2 | 讨厌场景 | 排除维度（15%权重） | 排除法缩小范围，被排除的维度不出现在3字码中 |
| Q3 | 能量来源 | I vs S 轴（15%权重） | 独处→I/R；社交→S/E；中性→不强烈倾向 |
| Q4 | 工作风格 | C vs A/I 轴（10%权重） | 规则明确→C；自由探索→A/I |
| Q5 | 兴趣驱动 | 主兴趣验证（20%权重） | 技术→R/I；帮人→S；创作→A；管理→E；整理→C |
| Q6 | 专业背景 | 现实约束（10%权重） | 专业对标维度映射，用户期望方向记录 |

## 五、综合推理权重分配

| 问题 | 权重 | 理由 |
|------|------|------|
| Q1（成就感） | 30% | 最能反映真实兴趣倾向，行为事件代表性强 |
| Q2（排除） | 15% | 排除法缩小范围，确定不合适的维度 |
| Q3（能量来源） | 15% | 区分内向/外向倾向，影响 I/S 轴判断 |
| Q4（工作风格） | 10% | 区分规则偏好 vs 自由探索，影响 C/A 轴 |
| Q5（主动钻研） | 20% | 行为验证，确认用户的自发兴趣方向 |
| Q6（专业背景） | 10% | 现实约束兜底，确保推荐方向在专业可行范围内 |

**权重调整规则**：
- 若某题用户跳过（skipped=true），该题权重均匀分配给其他题目
- 若某题置信度为"低"，该题权重减半
